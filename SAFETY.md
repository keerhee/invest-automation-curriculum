# 🛡️ 안전 수칙 (SAFETY.md)

> **읽지 않고 이 커리큘럼을 진행하지 마세요.** 이 문서를 읽는 데 5분이면 충분하지만, 한 번의 실수로 몇 년의 저축이 날아갈 수도 있습니다.

---

## 🚨 가장 중요한 3가지

### 1. 본 자료는 교육용입니다 — 투자 권유가 아닙니다

이 커리큘럼은 **AI/엔지니어링 학습 경험**을 제공하는 것이 목적입니다. 실제 투자 수익 창출은 보장하지 않으며, 본 자료의 어떤 내용도 특정 종목·전략의 매수·매도를 권유하지 않습니다.

### 2. 모든 실습은 Paper Trading(가상 자본)으로 진행합니다

- **Alpaca Paper Trading**: 가상 $100,000 자본
- **KIS 모의투자**: 가상 5천만원 자본
- **Binance Testnet**: 가상 자본

**실계좌 연동은 교과 과정에 포함되어 있지 않으며, 강력히 권장하지도 않습니다.**

### 3. 발생하는 모든 손실은 본인 책임입니다

자료 제작자(ZeroOneAI·원작자·기여자)는 이 자료를 사용하여 발생하는 **어떠한 금전적·정신적·기회비용상의 손실에 대해서도 책임을 지지 않습니다**. 코드 오류·API 변경·시장 변동·본인 판단 실수 등 모든 원인을 포함합니다.

## ⚠️ 실계좌 전환 시 경고

혹시라도 과정 이후 개인적으로 실계좌 전환을 고려한다면:

### 필독 자료
- [`docs/appendix/Appendix_E_Safety_Checklist.html`](docs/appendix/Appendix_E_Safety_Checklist.html)의 **"실계좌 전환 7대 검증"** 섹션
- 아래 위험 경고를 모두 이해했는지 자가 점검

### 실계좌 전환 위험 경고

```
□ 최소 3개월 이상 Paper Trading 무중단 운영 경험이 있는가?
□ Sharpe Ratio 1.0 이상을 실제로 달성했는가?
□ Max Drawdown을 -10% 이내로 유지했는가?
□ 시스템 버그·예외 케이스 0건 유지 기간이 있는가?
□ 잃어도 생활에 지장 없는 금액으로만 시작할 수 있는가?
□ Kill Switch(긴급 중단 방법)를 30초 내로 실행할 수 있는가?
□ 가족·지인에게도 Kill Switch를 공유했는가?
```

**하나라도 NO라면 실계좌 전환을 중단하세요.**

### 역사적 사고 사례

실제 자동매매 사고는 흔합니다:

- **2012 Knight Capital**: 배포 실수로 45분간 $460M 손실, 파산
- **2010 Flash Crash**: 알고리즘 매매 연쇄 작동으로 다우존스 9% 폭락
- 개인 투자자의 무한루프·계산 실수로 단일 야간에 수천만원~수억원 손실 사례 다수

**"나는 다를 것이다"는 가장 위험한 생각입니다.**

## 🔒 API 키 보안

### 절대 금지 사항

1. ❌ **GitHub 공개 리포에 `.env` 커밋**
   - 봇이 30초 내에 스캔해서 탈취합니다
   - `.gitignore`에 `.env` 반드시 포함 (이 리포는 기본 설정됨)

2. ❌ **Discord 스크린샷에 키 노출**
   - 강의 자료 · 질문 이미지에 API 키 모자이크 필수

3. ❌ **실전 키와 모의 키 혼용 저장**
   - Credential 이름에 "실전"/"모의" 명시
   - 색상으로 구분 (n8n에서 태그 기능 활용)

4. ❌ **하드코딩**
   - 코드에 `api_key = "sk-..."` 같은 형태로 직접 작성 금지
   - 환경변수 또는 n8n Credentials만 사용

### 권장 관행

1. ✅ **최소 권한 원칙**
   - Alpaca: Paper Trading 전용 키 사용 (실전 키와 분리 발급)
   - KIS: 모의투자 전용 앱 분리

2. ✅ **주기적 교체**
   - 3개월에 한 번 키 재발급
   - 수업 종료 후 발급한 키 폐기

3. ✅ **접근 로그 모니터링**
   - Anthropic Console, Alpaca Dashboard에서 이상 활동 확인

## 🧠 정신건강 주의

### 자동매매가 유발할 수 있는 문제

1. **강박 체크**: 몇 분마다 계좌를 확인하게 됨
2. **승부욕 과열**: 가상자본에서도 경쟁 심리 작동
3. **실전 전환 충동**: "모의에서 이겼으니 실전도 될 것"이라는 근거 없는 자신감
4. **수면 부족**: 24/7 암호화폐 시장 모니터링으로 수면 방해

### 건강한 학습을 위한 권장

- 📵 **Discord 알림을 특정 시간에만 켜기** (예: 평일 09~18시)
- ⏰ **하루 2시간 이상 과제에 쓰지 않기**
- 🛌 **밤 22시 이후 시스템 실행 금지** (자다가 주문 걱정 안 하기)
- 👥 **성과를 남과 비교하지 않기** (가상자본 수익률은 의미 없음)
- 🎯 **"얼마를 벌었나"가 아니라 "무엇을 배웠나"로 평가**
