# 💬 커뮤니티 규범 (Code of Conduct)

이 커리큘럼 커뮤니티는 **안전하고 포용적인 학습 환경**을 지향합니다.

---

## 🌱 우리가 지향하는 것

### ✅ 권장 행동

- **친절하고 건설적인 피드백** 제공
- 다양한 관점·배경·경험 존중
- 실수를 인정하고 배움의 기회로 삼기
- 개인이 아닌 **커뮤니티의 성장**에 집중
- 초보자에게 질문할 안전한 공간 제공

### ❌ 금지 행동

- 성적 언어·이미지·차별적 농담
- 인신공격 · 정치적 공격 · 모욕
- 공개 또는 비공개 괴롭힘
- 타인의 개인정보 공개 (주소·이메일·연락처·신원)
- 투자 권유 · 수익 자랑 · 펌프앤덤프
- 스팸 · 피싱 · 가짜 URL 링크
- 전문 환경에 부적절한 기타 행동

---

## 🎯 특별 주의사항 (투자 교육 커뮤니티 특성)

이 커뮤니티는 **금융·투자 교육**을 다루므로 추가 수칙이 있습니다:

### 💰 투자 조언 금지
- 다른 수강생에게 **"이 종목 사라"** 같은 권유 금지
- 본인 포지션 자랑 · 수익 인증 게시물 금지
- **"확실한 수익" 주장**은 즉시 경고 · 반복 시 제재

### 🔐 보안 준수
- **API 키 노출**은 즉시 해당 메시지 삭제 + 경고
- 다른 사람의 API 키 복사·저장·사용 금지
- 피싱 링크(인증 페이지 위장) 유포 시 **즉시 제재**

### 🧠 정신건강 배려
- 손실·실수에 대한 **조롱 금지**
- 자신의 실패담 공유는 환영 (배움의 기회)
- 심각한 정신건강 문제 언급 시 → 적절한 전문 도움 안내

### 📚 학습 우선
- 과제 답 통째로 공유 금지 (힌트·방향 제시만)
- 타인 제출물 무단 복사는 **표절**로 처리
- 커뮤니티 Q&A가 학습의 일부임을 존중

## 🚨 신고 방법

규범 위반을 발견하거나 경험했을 때:

### 1차 (경미한 경우)
- 해당 메시지에 직접 피드백 (예의 있게)
- Discord 채널에서 모더레이터 호출 (`@moderator`)

### 2차 (심각한 경우)
- 개인 메시지로 모더레이터에게 신고
- 스크린샷 + 상황 설명 첨부
- 모더레이터가 24시간 내 1차 대응

### 3차 (위급)
- 이메일: `conduct@example.com` (예시)
- 소속 기관이 있다면 기관의 공식 채널 동시 이용

**신고자의 신원은 비공개로 보호됩니다.**

---

## ⚖️ 제재 단계

| 단계 | 행동 | 결과 |
|:---:|---|---|
| 1차 경고 | 경미한 규범 위반 | 비공개 경고 · 메시지 수정/삭제 |
| 2차 경고 | 반복 위반 또는 중간 수준 | 공개 경고 · 일시 mute (24h) |
| 3차 제재 | 심각한 위반 | 1주~1개월 정지 |
| 영구 퇴출 | 극심한 위반 또는 반복적 악성 | 영구 BAN · 필요시 학교 통보 |

**즉시 영구 퇴출 사유**:
- 성적 괴롭힘·스토킹
- 타인의 금융 피해 유발
- API 키·개인정보 악용
- 피싱·사기 시도

---

## 🤝 모더레이터의 책임

- 공정하고 일관된 규범 집행
- 신고 접수 후 24시간 내 대응
- 당사자 양측의 의견 청취
- 제재 결정의 근거 기록 보존
- 커뮤니티의 심리적 안전감 유지

모더레이터도 규범을 지키지 않으면 제재 대상입니다.
