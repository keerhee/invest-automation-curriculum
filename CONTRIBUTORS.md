## 기여하고 싶으신가요?

1. [CONTRIBUTING.md](CONTRIBUTING.md)를 읽어주세요
2. 이슈 검색 또는 새 이슈 열기
3. Fork → Branch → PR
4. 리뷰 후 반영되면 이 파일에 추가됩니다!

**첫 기여**도 환영합니다. `good first issue` 라벨이 붙은 이슈부터 시작해보세요.

## 📜 기여자 권리

이 프로젝트에 기여하시면:

✅ **크레딧**: 이 문서에 이름 등재  
✅ **3회+ 기여 시**: Core Contributor 배지  
✅ **연간 우수 기여**: 공개 감사 + 기념품 (수업 있는 해)  
✅ **포트폴리오**: 본인 GitHub 프로필에서 해당 기여가 집계됨  

❌ **금전적 보상**: 없음 (비영리 교육 프로젝트)  
❌ **소유권**: 기여물은 프로젝트 라이선스(CC BY-NC 4.0)로 공개
