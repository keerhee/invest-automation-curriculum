Tool Introduction

n8n이란? 그리고 왜 투자에?

n8n은 "만약 ~이 발생하면, ~을 한다"는 규칙을 클릭으로 만드는 도구입니다. 코딩 없이 수백 개의 API·서비스를 연결해 반복 업무를 자동화합니다. 투자에서는 24시간 깨어있는 트레이더의 역할을 합니다.

쉽게 이해하기: 엑셀 매크로처럼 반복 작업을 자동화하는데, 엑셀 안이 아니라 거래소·뉴스·AI 사이를 연결합니다. 예: "BTC가 1시간 5% 하락하면 → Claude에게 뉴스 요약시키고 → Discord에 경고 발송"
🔌 트리거 (Trigger)
자동화를 시작시키는 신호.
투자 예시: 매일 오전 8시 · 주가 급락 Webhook · 관심종목 뉴스 RSS · 신규 공시(DART) 감지
⚙️ 액션 (Action)
트리거 이후 실행할 작업.
투자 예시: 시세 API 호출 · Google Sheets 기록 · Discord 알림 · 한투 API 주문 전송
🔀 조건 분기 (IF/Switch)
상황에 따라 다른 경로 선택.
투자 예시: RSI 30 미만 → 매수 알림 / 익절선 도달 → 매도 알림 / VIX 급등 → 포지션 축소 경고
🤖 AI Agent Node
Claude AI를 워크플로 안에 내장.
투자 예시: 공시 영향도 분석 · 실적 발표 요약 · 포트폴리오 리밸런싱 제안 · 매크로 지표 해석

n8n vs 기타 도구 비교

도구특징투자 자동화 적합성
n8n오픈소스, 셀프호스팅 가능, AI Agent 내장, Webhook 무제한높음 — API 키 로컬 보관
Zapier사용 편의성 최고, 클라우드 전용, 실행 횟수 제한보통 — 고빈도 자동화 불가
Python 직접 코딩완전 제어, 유연성 최고개발자 한정 — 학습 비용 큼

자동화 vs 에이전트 — 무엇이 다른가?

⚙️ 단순 자동화
정해진 규칙만 반복 실행.
예: "매일 9시 AAPL 가격을 시트에 기록" — 조건·분기는 있지만 판단은 없음
🤖 AI 에이전트
목표를 주면 스스로 판단.
예: "오늘 포트폴리오 리스크를 점검해" — 어떤 데이터를 볼지, 어떤 도구를 쓸지 AI가 결정

Outcomes

수료 시 보유 역량

01
n8n 워크플로 독립 설계·운영
시세 수집부터 알림 발송까지 — 4개 자산군(주식·코인·FX·선물)의 반복 업무를 코드 없이 자동화. Google Workspace·Discord·거래소 API를 자유롭게 연결해 본인 맞춤 파이프라인을 혼자 구축·수정 가능.
02
3D 분석 AI 에이전트 (가격·뉴스·차트)
프로 트레이더가 보는 3차원을 AI에게 동일하게 입력하는 에이전트. 가격·기술적 지표(1D) + 뉴스 센티먼트(2D) + TradingView 차트 이미지 + Claude Vision 패턴 판독(3D)을 종합해 매수·매도·보류 판단. 멀티에이전트 오케스트레이션으로 확장 가능.
03
24/7 자동 모니터링·알림 시스템
장 시작·마감 루틴 자동 발송·급등락 즉시 Discord 알림·실적 발표·FOMC 등 이벤트 대응 자동화. 멀티채널(Discord·텔레그램·Gmail) 동시 발송과 에러 자동 감지까지 무인 운영 체계 구축.
04
투자 리서치 AI 검색 시스템 (RAG)
"연준이 언제 금리 인하한다고 했지?", "이 ETF의 운용보수는?" — 증권사 리포트·FOMC 의사록·코인 백서·사업보고서를 Hybrid RAG로 검색해 출처 페이지까지 인용 답변. Discord 슬래시 커맨드로 언제든 질의.
05
한국투자증권 Open API 실전 연동
OAuth 토큰 발급·시세·잔고·주문 API 전체 숙달. 모의계좌에서 AI 시그널 기반 자동매매 파이프라인 완성. 안전장치(한도·승인·로그)까지 구비해 실계좌 전환 준비 완료.
06
본인 투자 시스템 PoC 1건 (포트폴리오로 활용 가능)
완성된 멀티에이전트 투자 자동화 시스템은 개발·금융 취업 포트폴리오로 직접 활용 가능. GitHub 공개·Notion 문서화·시연 영상까지 패키지화. "리서치 소요시간 2시간 → 5분" 등 정량적 성과 지표 포함.
