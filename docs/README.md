# 📚 학습 자료 전체 인덱스

이 폴더는 커리큘럼의 모든 학습 자료를 담고 있습니다.

## 📂 폴더 구조

```
docs/
├── 📖 lectures/    — 강의노트 HTML (10개)
├── 💻 labs/        — Python 실습 Jupyter (8개)
├── 📝 homework/    — 과제 워크북 HTML (9개)
└── 📚 appendix/    — 참고 부록 HTML (5개)
```

## 🗺️ 학습 순서

### 주차별 학습 방법 (권장)

각 주차마다 다음 3단계를 따릅니다:

1. **📖 강의노트 읽기** (60~90분)
   - `lectures/W#_Lecture_Notes.html`
   - 브라우저에서 직접 열기

2. **💻 실습 코드 돌려보기** (60~120분)
   - `labs/W#_Lab.ipynb`
   - Jupyter Notebook 또는 Google Colab
   
3. **📝 과제 완료** (3~5시간)
   - `homework/W#_Homework.html`
   - 필수 3개 + 심화 2개 (선택)

### 막혔을 때

- **📚 부록 A~E** 참조 (`appendix/`)
- Discord `#question` 채널
- [FAQ.md](../FAQ.md) 검색
